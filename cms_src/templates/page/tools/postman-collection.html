Postman Collection

Postman is a free REST Client that offers an easy-to-use interface for building and testing API calls. We provide a collection file (JSON format) that can be imported into Postman to define the full set of requests available in the Bluescape API so you can begin making requests and reviewing responses quickly.

In order to get started using Postman with the Bluescape API, follow these steps:

  1. Download the collection file to your computer
  2. Import the collection into Postman by using the Import button in the header toolbar or by selecting File > Import... and providing the downloaded file.
  3. Define the variables used in the collection (or replace the use of variables in urls and parameters with hard-coded values).
  4. Send your requests!
