User Authentication

In order for applications to access the Bluescape API they must be authorized by an existing Bluescape user. The User Authentication guide explains how to use your application's credentials to obtain this authorization.

Basic API Request

Once an application has receive an access token, it is free to start accessing the Bluescape API. The Basic API Request guide presents some example requests to get you started creating your application.

Listeners

Listeners are a powerful tool for monitoring Bluescape workspaces and responding to user actions by updating other systems. The Listeners guide describes in detail how listeners work and how to get started using them.
